Menstrudel is shown as a local Android app with sample period data already in place. The tour begins in normal app views so the reviewer can see the demo logs before the sharing feature appears. Those visible records are the source for the SMART Link, not a separate fixture.

In Data Management, SMART Link starts with local profile scope, date range, and category controls. The review emphasizes what will be included: logged bleeding and non-bleeding days, flow, symptoms, and pain levels. It also keeps clear boundaries: identity, predictions, period summaries, contraception, sexual activity, product details, and missing days are not part of this share. The preview is generated from the same snapshot used for the QR.

Create QR turns that reviewed snapshot into an encrypted live SMART Link. shlep stores ciphertext only; the decryption key remains in the URL fragment used by the viewer. Chromium opens the exact link, renders the clinical review after local decrypt, and Stop Sharing revokes the link so the previous QR no longer resolves.
