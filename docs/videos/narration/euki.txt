Euki is a privacy-first local tracker, and this recording starts after demo data has been loaded as normal Euki entries. We first tour the native experience: populated cycle views with logged bleeding, symptoms, products, and notes. The point is that SMART Link is built from the same local entries the user already sees, not from a detached test export.

In Cycle settings, SMART Link adds a controlled share review. There is no account selector because Euki keeps data local. The user checks date range and categories such as bleeding, flow, products, clots, emotions, body symptoms, and notes. Sexual activity, contraception, tests, appointments, predictions, and derived summaries are left out. The preview explains the selected facts and omitted areas before a QR exists.

Creating the SMART Link encrypts in the app and sends only ciphertext to shlep. QR, copy, native share, and Open Viewer all use the same live link. Desktop Chromium shows the clinician viewer resolving the link and decrypting in the browser. Stop sharing revokes it and returns Euki to a ready state.
