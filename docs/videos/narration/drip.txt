Drip is a local, single-profile Android period tracker, and this demo starts after synthetic sample cycles have already been loaded. The first screens are the normal app, not the share panel: we see populated cycle history, charting, and stats that the user would use day to day. From there the path moves into Settings, Data management, where SMART Link sits next to the existing import and backup tools.

The SMART Link screen is a review step, not a blind export. The user chooses a date range and switches for bleeding, flow, temperature, pain, mood, cervical observations, fertility signs, and notes. Predictions and unrelated sensitive fields stay out. The preview counts come from the same snapshot that will be shared.

When the QR is created, Drip encrypts the snapshot locally and uploads only compact ciphertext to shlep. The key stays in the link fragment. In Chromium, the doctor viewer opens that live link and decrypts locally. Stop Sharing revokes the managed link so the old QR stops working.
